
# ProgrammingPatterns

A collection of reusable .NET, WPF and MVVM components, patterns and examples gathered during learning and development.

## Purpose

This repository serves as a personal knowledge base for commonly used .NET patterns and implementations. It contains reusable code snippets, helper classes and examples that can be integrated into future projects.

The goal is to avoid rewriting frequently used functionality and to build a collection of proven solutions.
